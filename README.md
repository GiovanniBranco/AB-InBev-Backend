## Project setup

```bash
$ yarn install

$ cd docker; docker-compose up
```

Execute o comando para popular as tabelas:

```bash
$ yarn seed
```

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Run tests

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
